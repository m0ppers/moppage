Babylon.js - 3D Web Game Development made easy

Andreas Streichardt (@m0ppers)

Babylon.JS

  • Open Source WebGL Engine
  • Specifically designed for Game Development
  • Physics Support (via cannon.js/Oimo.js)
  • Offline Support (IndexedDB)
  • Multiple Input devices (Gamepad, Touchpad, Mouse etc)
  • Multiple Output devices (WebVR, Occulus) :O
  • Intended to be simple to use
  • Written in TypeScript
  • Done by some guys @Microsoft

State of WebGL

  • With iOS 8 released EVERY notable platform has WebGL support
  • WebGL 2 is being worked on
  • Based on OpenGL ES 2.0 (mobile OpenGL, which is supported on every smartphone)
  • Many apps are done using HTML5, expect the same for games now

Babylon.JS is easy to use / get you started

  • Easy scene.pick support
  • Awesome blender scene importer (create a WebGL Demo in 5 minutes)
  • Animation system
  • Integrated assets loader
  • DebugLayer
  • Sound
  • Procedural Textures
  • Physics engine support

Babylon.JS is designed for performance

  • Instancing
  • Static drawing
  • Mesh merging
  • Incremental scene loading
  • SIMD.js (ES7) (probably Spartan showcase ;) )

Hands on!
